# Khushi Chauhan — Modeling Portfolio

A modern, responsive fashion and commercial modeling portfolio for Khushi Chauhan. Built with React, Vite, Express and custom CSS.

## Local development

```bash
npm install
npm run dev
```

The website runs at `http://localhost:5173` and the booking API at `http://localhost:3001`.

## Production

```bash
npm run build
npm run preview
```

The Vite base path is configured for GitHub Pages at `/khushi-model-portfolio/`. The Express API must be hosted separately for live booking submissions.
